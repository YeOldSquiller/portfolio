  • The background of the game is located in a cave because Fishapod is the first evolution of fish to have legs.

    Features

    Feature # Short Name Description Status/Functionality
    1 Collision Mask Collision masks are in this game to create a spot on the sprite in which the player gets hit, this is important to the game because without them it would make it harder and more cheap to get by the enemies. There are collision masks on multiple sprites such as the player, and the enemies.
    2 Movement We have this feature in order to even play the game, without movement you would just be staring at a screen. But adding movements like jumping and double jumping are added to avoid obstacles. Left arrow keys moves left and right arrow key moves right. Space bar is to jump and the up arrow key is to double jump.
    3 Animation It makes the game look better and more neat, in general more professional. Dying animation, jumping animation, falling animation, walking animation and the fly moving animation.
